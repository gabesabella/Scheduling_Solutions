Title: Scheduling Solutions

Purpose: Scheduling solutions is an appointment management application. It enables users to create and manage appointments of customers
from differing timezones. Username and password authentication is used to enter the program. This application uses an SQL database for storage.
It also generates reports that provide a variety of information for the users.
